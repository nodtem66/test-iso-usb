## A test for isochronous usb
3 programs
- **test_iso**
    libusb1.0 (linux) for isochronouse endpoint 0x83
- **test_iso_libusbk**
    libusbk (window) for isochronouse endpoint 0x83
- **test_libusbk**
    libusbk (window) for interrupt endpoint 0x81

### Installation
##### Linux
    make
    
##### Window
    Codelite Editor > open USB.workspace
